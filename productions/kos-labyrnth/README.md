# Ko's Labyrnth, An Alchemist's Labor of Love

Primary production root for this title.

## Structure

- `preproduction/`: concept, scope, milestones
- `design/`: narrative, systems, economy, quest design docs
- `content-packs/`: canonical game data packs (characters, quests, items, flags)
- `scenes/`: scene source and compiled scenegraph assets
- `scripts/`: runtime plans and scripting artifacts specific to this production
- `art/`: visual assets and source files
- `audio/`: music, SFX, VO staging assets
- `builds/`: packaged playable outputs

## Conventions

- Keep canonical IDs stable once published to playtest/beta.
- Treat runtime plans as deterministic test fixtures.
- Keep this folder as the source-of-truth production boundary for the title.
